## Project description
Brain Games is a JavaScript project made from scratch for Node.js runtime environment. 'Brain Games' includes 
program to greet the user and five games to train user`s calculation skills.


## Project set up

</ol>


## How to play games
There are six CLI commands:
<ol>
<li>brain-games</li>
<li>brain-even</li>
<li>brain-calc</li>
<li>brain-gcd</li>
<li>brain-progression</li>
<li>brain-prime</li>
</ol>
<p>Use one of the commands to run the respective program in the terminal.</p>
<p>Answer its questions. </p>
<p>Win.</p>
